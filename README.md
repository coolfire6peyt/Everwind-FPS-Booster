# ⚡ FPS Booster for Everwind

> A pure performance optimization mod for Everwind. No visual downgrades, no removed assets — just more FPS.

---

Tired of frame drops and stuttering? **FPS Booster** improves your game's performance **without sacrificing visual quality**.

---

## ✅ What this mod does
- Optimizes how the game processes and renders scenes
- Reduces unnecessary background load on CPU/GPU
- Provides smoother and more stable frame rates
- Improves overall performance on both low-end and high-end systems

## 🚫 What this mod does NOT do
- ❌ Does **not** remove grass, trees, particles, or any visual elements
- ❌ Does **not** lower texture quality or resolution
- ❌ Does **not** cut or strip any game content
- ❌ Does **not** alter the way the game looks in any way

---

## 🎯 The goal is simple
**Same visuals. Better performance.**

This mod works under the hood — optimizing the way the game runs, not the way it looks. Everything you see in vanilla Everwind stays exactly the same.

---

## 📥 How to use
1. Download the mod
2. Extract the files to the game folder
3. Run **`Everwind-FPS-Boost.exe`**
4. Launch the game and enjoy the extra frames

> ⚠️ Make sure to run **Everwind-FPS-Boost.exe** before or alongside the game for the optimization to take effect.

---

## 🔧 Compatibility
- Works with most other mods
- No known conflicts
- Safe to install and uninstall at any time

---

> *Play Everwind the way it's meant to look — just faster.*
